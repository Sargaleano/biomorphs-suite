In 1986, Richard Dawkins wrote a program that changed how we think about biological complexity forever. His **biomorph** experiment — originally coded in Pascal and run on a Macintosh — showed that intricate, organic-looking structures could emerge from a handful of integer genes and a recursive algorithm, with no designer, no purpose, no teleological guidance. Just cumulative selection.
 
**Forty years later**, that landmark experiment remains one of the most elegant and pedagogically powerful computational demonstrations of evolutionary theory. To mark four decades since the concept was born, we present **Biomorphs Suite**: a faithful web-based reimplementation of Dawkins's original 9-gene model, developed for educational use at the Universidad Distrital Francisco José de Caldas. The suite expands on the original with a simplified 5-parameter interface for rapid exploration, a curated library of classic evolved forms, and a companion manual sculpting mode.
 
**No installation. No server. Just open and evolve.**

## What are Biomorphs?
 
Biomorphs are recursive, tree-like creatures defined by a vector of 9 integer genes. Each gene controls a geometric property — branch angles, lengths, scaling, recursion depth. The drawing algorithm is deceptively simple: a binary tree where each branch spawns two children, offset by direction vectors derived from the genome.
 
The magic happens through **cumulative selection**: the user clicks one offspring per generation, acting as the selector. Within a dozen generations, forms resembling insects, ferns, bats, spiders and crabs emerge — not because they were designed, but because small changes accumulate over time.
 
```
Genome:  [-6, -6, -6, -8, -4, 0, 4, 8, 5]   ← Dawkins's archetype
          g0  g1  g2  g3  g4  g5 g6 g7 g8 (depth)
```

## Features
 
### 🔬 Biomorph Evolver
- **Generational selection** — click any offspring to breed it; watch morphology drift, converge, or leap
- **Brood sizes** — 1×3, 3×3, 1×5, 5×5 offspring grids
- **Mutation controls** — *Turbulence* (how many genes change) and *Leap* (how much each gene changes)
- **Two genome modes** — Friendly (5 intuitive parameters) or Original (raw 9-gene Dawkins model)
- **🦴 Fossil record** — every selected parent is automatically archived; rewind to any ancestor
- **⭐ Personal gallery** — star your favourite creatures and return to them any time
- **📖 Classics library** — 9 curated starting genomes: Cricket, Fern, Spider, Bat, Pyramid, Scorpio, Beetle, Tree, Crab
- **Save / Load** — export and restore broods and galleries as JSON files
- **📸 Screenshot** — download the full brood grid as a PNG image
### 🎨 Biomorph Creator
- **Precise genome sculpting** with real-time canvas preview
- **Friendly mode** — 5 high-level parameters (Complexity, Spread, Height, Taper, Sprawl) projected to 9 genes via a mathematical mapping
- **Original mode** — direct access to all 9 Dawkins genes, full ±9 integer range
- **Export to Evolver** — design a creature here, seed it into the Evolver gallery with one click
- **📸 Screenshot** — save your creation as a PNG
### 🌐 Technical
- Pure **HTML5 + CSS3 + JavaScript ES6+** — zero dependencies, zero frameworks
- Renders via the browser's native **Canvas API 2D**
- Fully **client-side** — no server, no backend, no data collection
- Works in any modern browser (Chrome, Firefox, Safari, Edge)

## The 9-Gene Model
 
| Gene | Name | Effect | Range |
|---|---|---|---|
| g0 | Inner spread | Horizontal displacement of innermost branches | −9 to 9 |
| g1 | Mid spread | Horizontal displacement of mid-level branches | −9 to 9 |
| g2 | Outer spread | Horizontal displacement of outermost branches | −9 to 9 |
| g3 | Trunk length | Vertical extent of the central trunk | −9 to 9 |
| g4 | Inner height | Vertical extent of inner branch tier | −9 to 9 |
| g5 | Mid height | Vertical extent of mid branch tier | −9 to 9 |
| g6 | Outer height | Vertical extent of outermost tier | −9 to 9 |
| g7 | Sprawl | Overall horizontal scaling factor | −9 to 9 |
| g8 | Depth | Recursion depth — branching levels | 3 to 9 |
 
The **Friendly mode** maps 5 intuitive parameters `[Cx, Sp, Ht, Ta, Sw]` to the full 9-gene space via the following transformation:
 
```js
-----------------------------------------------------------------
t  = Ta / 9
-----------------------------------------------------------------
g0 = round(Sp)      g1 = round(Sp×0.7)    g2 = round(Sp×0.4)
-----------------------------------------------------------------
g3 = Ht             g4 = Ht
-----------------------------------------------------------------
g5 = round(Ht×(1+t))                g6 = round(Ht×(1+t)×(1+t×0.5))
-----------------------------------------------------------------
g7 = Sw             g8 = Cx
-----------------------------------------------------------------
```

## Getting Started Locally
 
No build step required. Clone and open:
 
```bash
git clone https://github.com/sargaleano/biomorphs-suite.git
cd biomorphs-suite
open index.html          # macOS
# or: start index.html   # Windows
# or: xdg-open index.html  # Linux
```
 
Or just download the ZIP from the green **Code** button above and open any `.html` file directly in your browser.

## Creator → Evolver Workflow
 
Design a precise starting creature and then evolve it:
 
1. Open the **Creator** and sculpt your genome with the sliders
2. Click **⏬ Export biomorph** — saves `biomorph-gallery.json`
3. Open the **Evolver** (or click 👽)
4. Go to the **⭐ Gallery** tab and click **⏫** to load the file
5. Click your creature in the gallery to set it as parent
6. Start selecting offspring — evolution begins

## Academic Context
 
This project was developed at the **Universidad Distrital Francisco José de Caldas** (Bogotá, Colombia) as an educational tool for teaching concepts of evolutionary computation, emergent complexity, and human-in-the-loop selection. It is intended for use in undergraduate and postgraduate courses in computer science, artificial intelligence, and computational biology.
